# Basketball Stats Manager - Version Management

## מערכת גרסאות

הפרויקט משתמש במערכת גרסאות Semantic Versioning (SemVer) עם מספר BUILD אוטומטי.

### פורמט הגרסה
```
vMAJOR.MINOR.PATCH+BUILD
```

**דוגמה:** `v1.0.0+20250117.1430`

### רכיבי הגרסה

- **MAJOR** - שינויים גדולים שאינם תואמים לאחור
- **MINOR** - תכונות חדשות תואמות לאחור  
- **PATCH** - תיקוני באגים תואמים לאחור
- **BUILD** - מספר BUILD אוטומטי (YYYYMMDD.HHMM)

### מספר BUILD

מספר ה-BUILD נוצר אוטומטית על בסיס התאריך והשעה הנוכחיים:
- **תאריך**: YYYYMMDD (שנה-חודש-יום)
- **שעה**: HHMM (שעה-דקה)

**דוגמה:** `20250117.1430` = 17 בינואר 2025, 14:30

## שימוש במערכת הגרסאות

### תצוגת הגרסה
הגרסה מוצגת בכותרת האתר ומתעדכנת אוטומטית.

### עדכון גרסה ידני
```javascript
// הגדלת גרסה ראשית (MAJOR)
window.versionManager.incrementVersion('major');

// הגדלת גרסה משנית (MINOR)  
window.versionManager.incrementVersion('minor');

// הגדלת גרסת תיקון (PATCH)
window.versionManager.incrementVersion('patch');
```

### קבלת מידע על הגרסה
```javascript
// קבלת מחרוזת הגרסה המלאה
window.versionManager.getVersionString();

// קבלת מידע מפורט על הגרסה
window.versionManager.getVersionInfo();
```

## היסטוריית גרסאות

### v1.3.0 - כלי מיזוג שחקנים (26/10/2025)
- **תכונה חדשה**: כלי מיזוג שחקנים כפולים
- זיהוי אוטומטי של שחקנים עם שמות דומים
- בחירה ידנית עם checkboxes
- סנכרון קריטי למניעת אי-עקביות בנתונים
- אלגוריתם Levenshtein Distance לזיהוי שגיאות כתיב
- תיקון באגים: הוספת אתחול `initPlayerMergeTool()`

### v1.2.0 - ניתוח משחק מתקדם (18/10/2025)
- מודול ניתוח משחק מתקדם
- Four Factors (eFG, TOV%, ORB%, FT Rate)
- Offensive/Defensive Ratings
- ניתוח פרופיל קליעה ויעילות מצבית

### v1.1.0 - הכנה למשחק ומערכת זהות שחקנים
- מערכת הכנה למשחק עם נרטיבים בעברית
- מערכת זהות שחקנים חדשה (UUID, canonical keys)
- כלי מיזוג קבוצות
- כלי ניהול העברות

### v1.0.0 - גרסה ראשונית
- מערכת ניהול סטטיסטיקות כדורסל
- ייבוא נתונים מ-IBBA
- ניהול קבוצות ושחקנים
- שמירה למסד נתונים מקומי
