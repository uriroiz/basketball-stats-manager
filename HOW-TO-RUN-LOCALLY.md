# 🏀 איך להריץ את האתר מקומית

## ⚡ הדרך הכי מהירה

1. **לחץ פעמיים על הקובץ:** `start-server.bat`
2. **פתח בדפדפן:** http://localhost:8000
3. **זהו! האתר רץ** ✅

---

## 📝 הסבר

בגלל מגבלות CORS, האתר לא יכול לטעון נתונים מ-API כשהוא רץ מ-`file://`.
הפתרון: להריץ HTTP server מקומי פשוט.

---

## 🛑 איך לעצור את ה-Server

1. לחץ בחלון הטרמינל
2. לחץ `Ctrl + C`
3. או פשוט סגור את החלון

---

## 🔧 אפשרויות נוספות

### Python (אם קיים)
```bash
python -m http.server 8000
```

### Node.js (אם קיים)
```bash
npx http-server -p 8000
```

### PHP (אם קיים)
```bash
php -S localhost:8000
```

---

## 🌐 פורטים אחרים

אם פורט 8000 תפוס, נסה:
```bash
python -m http.server 8080
```
ופתח: http://localhost:8080
