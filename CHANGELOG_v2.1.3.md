## 🧪 איך לבדוק

### בדיקה: Turnover Creators / Steal Masters
```
1. בחר קבוצה עם חטיפות גבוהות (SPG >= ממוצע + 2)
2. ודא שמוצג: "חטיפות למשחק" (ולא "גניבות")
3. ודא שמוצג: "(מקום X בליגה בחטיפות)"
4. בדוק ניסוח:
   - מקום 1: "מלכת החטיפות"
   - מקום 2: "מצטיינת בחטיפות"
   - מקום 3+: "חוטפת כדורים"
```

### בדיקה: League Leader
```
1. בחר קבוצה במקומות ראשונים בחטיפות (SPG)
2. ודא שמוצג: "🏃 מקום X בחטיפות (Y.Y)"
3. ודא שלא מופיע "בגניבות"
```
